# 🚢 Titanic EDA — Exploratory Data Analysis

## 📌 Project Overview
Exploratory Data Analysis on the Titanic dataset to find 
patterns in passenger survival using Python.

## 📊 Dataset
- 891 passengers, 12 columns
- Source: Kaggle Titanic Dataset

## 🔍 Key Findings
- Overall survival rate: ~38%
- Female survival rate: ~74% vs Male: ~19%
- 1st Class survival: 63% vs 3rd Class: 24%
- Children (0-10) had higher survival rate
- Higher Fare = better survival chance (corr: +0.26)

## 📈 Visualizations
1. Survival Count Overview
2. Survival by Gender
3. Survival by Passenger Class
4. Age Distribution by Survival
5. Correlation Heatmap
